Frequently Asked Questions

Our RAG system combines the power of LLaMA 3.3 70B with efficient document retrieval. When a user asks a question:
  1. The query is converted to embeddings using AWS Bedrock
  2. Relevant documents are retrieved from MongoDB
  3. LLaMA generates a response using the retrieved context

To optimize performance:
  • Adjust temperature and top-p settings in Configuration
  • Fine-tune chunk sizes for document processing
  • Monitor and analyze user feedback
  • Regularly update your knowledge base

In the Data Sources section, you can:
  • Add new websites for crawling
  • Upload and process PDF documents
  • Set crawling schedules and rules
  • Monitor ingestion status
